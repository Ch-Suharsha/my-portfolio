# Suharsha Cheedalla - Portfolio

A set of professional, monochrome portfolio designs built with vanilla HTML, CSS, and JavaScript.

## Versions
- **Elegant** (`index.html`): The primary, clean, and sophisticated version.
- **Luxury** (`luxury.html`): A bold, dark-themed alternative.

## Deployment
This static site is optimized for one-click deployment to [Vercel](https://vercel.com/new). No build step is required—Vercel will serve `index.html` natively.
